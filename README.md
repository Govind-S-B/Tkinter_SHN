# Tkinter_SHN
Tkinter Project for SHN ( December 3)

A simple mood tracker/journaling application built on Tkinter
<br>
Trying to push whats possibe with Tkinter in terms of modern UI

## How to Run
navigate to project directory
run `python main.py`
